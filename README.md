
## antd-easy

> 基于antd库进行再次封装，使用更便捷，使表单错误提示及校验更友好


### 准备，clone 并安装依赖

```
git clone git@github.com:tt-ghost/antd-easy.git
cd antd-easy
npm i
```

### 启动

#### 启动服务端，用于返回示例文件源码，默认启动 *3100* 端口

```npm
npm run server
```

#### 执行以下代码运行前端代码，成功后访问 [http://localhost:9100](http://localhost:9100)

```npm
npm start
```
